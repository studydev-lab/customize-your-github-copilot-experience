# 📘 Assignment: Python Data Visualization

## 🎯 Objective

Students will learn how to turn raw data into clear charts and graphs using Python visualization libraries. They will practice creating and customizing visualizations with either matplotlib or plotly.

## 📝 Tasks

### 🛠️ Build Basic Charts

#### Description
Load sample data and create core chart types to communicate trends, comparisons, and relationships.

#### Requirements
Completed program should:

- Load tabular data from a CSV file using pandas
- Create at least 3 chart types (bar chart, line chart, and scatter plot)
- Add clear titles and axis labels to every chart


### 🛠️ Customize and Present Insights

#### Description
Improve chart readability and explain what the visuals show.

#### Requirements
Completed program should:

- Use at least 2 styling/customization techniques (color palette, grid, marker style, legend, theme)
- Save at least 2 charts to files (PNG for matplotlib or HTML for plotly)
- Write 2 short insight statements based on the generated charts
