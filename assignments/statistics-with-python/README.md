# 📘 Assignment: Statistics with Python

## 🎯 Objective

Students will use Python to analyze real-world tabular data and compute core statistical measures. They will practice data analysis workflows with pandas and numerical/statistical calculations with numpy.

## 📝 Tasks

### 🛠️ Data Exploration with pandas

#### Description
Load a CSV dataset and perform initial exploration to understand its structure, data quality, and key patterns.

#### Requirements
Completed program should:

- Load a CSV file into a pandas DataFrame
- Display dataset shape, column names, and the first 5 rows
- Identify missing values per column and output a brief summary


### 🛠️ Statistical Analysis with numpy

#### Description
Select at least one numeric column and compute descriptive statistics and simple comparisons.

#### Requirements
Completed program should:

- Compute mean, median, standard deviation, minimum, and maximum using numpy
- Calculate at least one percentile (for example 25th or 75th percentile)
- Print 2 short insight statements based on the computed results
